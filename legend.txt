.venv/ is for python virtual environment
app/ is the flask app package
    routes/ is blueprints for all the endpoints
    models/ is the sqlalchemy orm models
    services/ is business logic like auth, scoring, etc
    __init__.py is the app factory
config.py is the app configuration like env vars, db uri etc
db.py is the sqlalchemy instance
run.py is the entrypoint to launch server
requirements.txt is pinned python dependences
